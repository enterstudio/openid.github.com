Change Log


Changes in openid.server.server.CheckIDRequest.answer
  • Version 2.0 deprecates server_url and adds claimed_id.
Changes in openid.server.server.OpenIDResponse.whichEncoding
  • 2.1.0 added the ENCODE_HTML_FORM response.
Changes in openid.server.server.Server.__init__
  • op_endpoint is new in library version 2.0. It currently defaults to None for compatibility with earlier versions of the library, but you must provide it if you want to respond to any version 2 OpenID requests.
Changes in openid.store.interface.OpenIDStore
  • Version 2.0 removed the storeNonce, getAuthKey, and isDumb methods, and changed the behavior of the useNonce method to support one-way nonces. It added cleanupNonces, cleanupAssociations, and cleanup.
Changes in openid.store.interface.OpenIDStore.useNonce
  • In earlier versions, round-trip nonces were used and a nonce was only valid if it had been previously stored with storeNonce. Version 2.0 uses one-way nonces, requiring a different implementation here that does not depend on a storeNonce call. (storeNonce is no longer part of the interface.)
